# RKSI_shedule
## Приложение с множеством возможностей для удобного просмотра расписания
PS: Пока не решил буду ли публиковать код, но функционал понемногу буду расписывать.

Разделы:
  Главная
  Настройки
  Расписания
  Планшетка

### Главная
Отображение расписания с сайта + изменений в расписании. Имеется 3 режима отображения: 1/3/6 дней на экан. Изменяется с помощью жестов увеличения/уменьшения.
Имеется 2 списка (с преподами и с группами) выбрав элемент - его расписание отобразится вместе с изменениями, если были внесены.

Менять расписание можно вставив сообщение от черепахи в выпадающем меню во вкладке "черепаха".
Если нажать на пару - отобразится полная информация о ней (номер, время, название пары, аудитория, преподаватель/группа). В выпадающе менб можно редактировать пару или удалить её.
  Во вкладке "редактировать" можно поменять название пары (предоставляются пары группы), аудиторию, преподавателя и номер пары.

### Настройки
Выбор расписания по умолчанию, можно задать ссылку на папку с планшеткой и изменить имя преподавателя/название группы на определённом предмете опеделённой группы / преподавателя.
